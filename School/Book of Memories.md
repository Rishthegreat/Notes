#### Derivaties of Trig Functions
| Function  | Derivative        |
| --------- | ----------------- |
| $\sin{x}$ | $\cos{x}$         |
| $\cos{x}$ | $-\sin{x}$        |
| $\tan{x}$ | $\sec^2{x}$       |
| $\cot{x}$ | $-\csc^2{x}$      |
| $\sec{x}$ | $\sec{x}\tan{x}$  |
| $\csc{x}$ | $-\csc{x}\cot{x}$ |

#### Derivaties of Inverse Trig Functions
| Function       | Derivative               |
| -------------- | ------------------------ |
| $\sin^{-1}{x}$ | $\frac{1}{\sqrt{1-x^2}}$ |
|                |                          |
